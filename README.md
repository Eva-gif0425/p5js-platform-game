# p5.js Platform Game - Test Paper Chase

## 專案簡介
使用 p5.js 製作的 2D 平台跳躍遊戲，玩家需要控制角色在平台之間移動並收集考試卷來獲得分數，分數到達100即結束遊戲。

## 功能
- 玩家左右移動與跳躍
- 平台碰撞判斷（包含移動平台）
- 道具收集系統（考試卷）
- 分數計算與遊戲結束機制
- 主選單與重新開始

## 使用技術
- JavaScript (p5.js)
- HTML / CSS

## 操作方式
- ← →：左右移動
- 空白鍵 / ↑：跳躍
